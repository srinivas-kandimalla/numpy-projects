# NumPy Projects

This repository contains my practice projects built using NumPy while learning data analysis and numerical computing with Python.

## Projects

### 1. House Price Analysis
Analyzes house sizes and prices and predicts house price based on square feet.

### 2. Student Marks Analysis
Analyzes student marks across multiple subjects and calculates statistics such as averages and top-performing students.

### 3. Sales Data Analysis
Performs analysis on sales data to calculate totals, averages, and identify best-performing products.

### 4. Matrix Calculator
Implements common matrix operations such as addition, subtraction, and multiplication using NumPy.

### 5. Random Data Analysis
Generates random datasets using NumPy and performs statistical analysis.

### 6. NumPy Linear Regression
Implements a simple linear regression model using NumPy to understand machine learning basics.

## Technologies Used

- Python
- NumPy
